## 应用描述

### 一句话简介（短描述，≤80字）

AI 潮汐：每周精选 GitHub AI/ML 开源项目，AI 生成中文摘要，支持跨设备收藏同步。

### 完整描述（≤4000字）

**AI 潮汐** 是一款面向开发者的开源项目追踪工具，帮助你高效发现 GitHub/GitLab/Gitee 上最热门的 AI/ML 项目。

#### 核心功能

📊 **每周 Trending 精选**
- 自动抓取 GitHub、GitLab、Gitee 三大平台 Trending 页面
- 支持按编程语言筛选（Python/Go/Rust/TypeScript 等 20+ 语言）
- 时间范围：今日/本周/本月

🤖 **AI 智能摘要**
- LLM 自动生成项目中文摘要和核心要点
- 无需逐篇阅读 README，快速了解项目价值
- 支持 DeepSeek/OpenAI/Claude/Qwen 多种模型

⭐ **收藏与跨设备同步**
- 注册账号后收藏项目，数据云端同步
- 桌面端 (Windows Tauri)、Android、鸿蒙三端一致
- 换设备登录同一账号，收藏列表自动恢复

🔔 **周报推送通知**
- 每周一自动推送新一期 Trending 周报
- Android FCM / 鸿蒙原生通知栏
- 点击通知直达周报详情

📈 **Star 趋势图表**
- 查看项目 30 天 Star 增长曲线
- 发现正在快速崛起的新项目

📡 **离线可用**
- 断网时自动使用本地缓存数据
- 地铁通勤无网络也能浏览

## English Listing (for non-Chinese users)

### Short Description (≤80 chars)

AI Tide: Weekly curated GitHub Trending AI/ML repos with AI summaries & cross-device sync.

### Full Description

**AI Tide (DevPulse)** is a developer tool that helps you discover the hottest open-source AI/ML projects across GitHub, GitLab, and Gitee.

- 📊 Weekly trending repos with 20+ language filters
- 🤖 AI-generated Chinese summaries & key points
- ⭐ Bookmark repos & sync across Windows/Android/HarmonyOS
- 🔔 Push notifications for new weekly reports
- 📈 Star history trend charts
- 📡 Offline-ready with local caching

Free forever. No ads. Open source.
