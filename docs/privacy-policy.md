# DevPulse（AI 潮汐）隐私政策

> **最后更新日期**：2026-05-28  
> **生效日期**：2026-05-28

## 1. 引言

DevPulse（"我们"、"我们的"）深知个人信息对您的重要性。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。使用 DevPulse 即表示您同意本隐私政策。

## 2. 我们收集的信息

### 2.1 您主动提供的信息

| 信息类型 | 说明 | 用途 |
|---------|------|------|
| **邮箱地址** | 注册时提供 | 账号标识、登录凭证 |
| **显示名称** | 可选，注册时提供 | 在应用界面中展示 |
| **密码** | 注册/登录时提供 | 经 bcrypt 加密存储，我们无法获知明文 |

### 2.2 我们自动收集的信息

| 信息类型 | 说明 | 用途 |
|---------|------|------|
| **收藏数据** | 您在应用中收藏的项目列表 | 跨设备同步 |
| **推送偏好** | 通知开关设置 | 控制推送行为 |
| **FCM Token** | Android 设备推送标识符 | 发送推送通知 |

### 2.3 我们不收集的信息

- ❌ 精确地理位置
- ❌ 设备通讯录
- ❌ 相册/相机内容
- ❌ 生物识别信息
- ❌ 浏览历史（非 DevPulse 内的网页）

## 3. 信息的使用方式

我们收集的信息仅用于以下目的：

1. **提供核心服务**：跨设备收藏同步、周报推送
2. **改善服务质量**：分析使用趋势以优化功能
3. **安全保障**：检测和防止欺诈、滥用行为

我们**不会**：
- 将您的个人信息出售给第三方
- 使用您的数据投放广告
- 基于您的数据构建用户画像用于商业目的

## 4. 数据存储与安全

### 4.1 存储位置

您的数据存储在 Supabase 提供的 PostgreSQL 数据库中（加密传输，TLS 1.3）。

### 4.2 安全措施

| 措施 | 说明 |
|------|------|
| **密码加密** | bcrypt（cost factor=12）哈希，不可逆 |
| **传输加密** | 全链路 HTTPS/TLS 1.3 |
| **Token 安全** | JWT access token 24h 有效期，refresh token 7d 有效期 |
| **数据库加密** | PostgreSQL 静态数据加密 |

### 4.3 数据保留

- 账户数据：保留至账户删除
- 收藏数据：保留至账户删除或手动取消收藏
- FCM Token：保留至退出登录或手动注销

## 5. 您的权利

根据适用法律（包括《个人信息保护法》），您享有以下权利：

| 权利 | 操作方式 |
|------|---------|
| **访问权** | 在应用"设置"页面查看个人信息 |
| **更正权** | 通过应用内功能修改显示名称和推送偏好 |
| **删除权** | 发送邮件至 devpulse@proton.me 申请删除账户及所有关联数据 |
| **撤回同意** | 在"设置"中关闭推送通知；退出登录即停止数据同步 |
| **数据可携带** | 发送邮件申请导出个人数据（JSON 格式） |

我们将在收到请求后 **30 个自然日内** 处理完毕。

## 6. 第三方服务

DevPulse 使用以下第三方服务：

| 服务 | 用途 | 隐私政策 |
|------|------|---------|
| **Supabase** | 数据库托管 | https://supabase.com/privacy |
| **Firebase Cloud Messaging** | Android 推送通知 | https://firebase.google.com/support/privacy |
| **DeepSeek/OpenAI/Claude** | AI 摘要生成 | 仅传输项目公开信息（README），不含用户数据 |
| **Render/Railway** | 后端 API 托管 | 不存储用户数据 |

## 7. 儿童隐私

DevPulse 不面向 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。如发现误收集，将立即删除。

## 8. 隐私政策更新

我们可能会不时更新本隐私政策。重大变更将通过以下方式通知：
- 应用内弹窗提示
- 注册邮箱通知（如已提供）

建议定期查阅本页面以了解最新版本。
