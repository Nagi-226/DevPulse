# DevPulse（AI 潮汐）增量产品需求文档

> **版本范围**：0.1.2 → 0.2.0（Phase 2：移动端）  
> **基线版本**：0.1.0（Phase 1 完成，Tauri 桌面 MVP 可用）  
> **语言**：中文  
> **技术栈**：Python FastAPI + React 18 + Tauri 2.x + Zustand + Tailwind CSS + SQLite  
> **文档日期**：2026-05-28

---

## 1. 原始需求复述

在 0.1.0 基线之上，逐步补齐数据质量（Stars/Forks）、激活 LLM 摘要管线、增加桌面驻留（系统托盘）、搜索、收藏管理、趋势图表、离线缓存、推送通知、Capacitor 移动端基础，最终交付 Android APK 发布版。

---

## 2. 产品目标

| # | 目标 | 衡量标准 |
|---|------|---------|
| G1 | **数据完整可信** | 每个 Trending 卡片展示 Total Stars / Stars Since / Forks 三项真实指标，无零值缺失 |
| G2 | **AI 摘要可用** | 每个项目详情页展示中文摘要 + 3 个关键点 + 标签，LLM 管线在爬虫完成后自动触发 |
| G3 | **桌面体验闭环** | 关闭窗口驻留托盘、支持搜索/收藏/图表/推送通知，断网时展示缓存数据 |
| G4 | **Android MVP 发布** | `npx cap sync android` 成功，Gradle 构建产出签名 APK，真机可运行 |

### 5.3 关键交互细节

**收藏操作**：
```
详情页 ⭐ 按钮 → 点击 → 星标填充 + Toast "已收藏"
                   → 再次点击 → 弹出确认 "确定取消收藏？"
                              → 确认 → 星标空心 + Toast "已取消"
```

**搜索交互**：
```
输入关键词 → 300ms 防抖 → 调用 API → 列表替换为搜索结果
输入为空 → 恢复默认 Trending 列表
无匹配 → 显示 "未找到匹配的项目" + 清空按钮
```

**离线模式**：
```
断网 → 顶部滑入黄色 Banner → 列表展示缓存数据
恢复网络 → Banner 滑出 → 自动刷新 → 列表更新
```
