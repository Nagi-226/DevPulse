# AppGallery 商店描述

## 应用名称
AI 潮汐 DevPulse

## 应用简介 (67字以内)
GitHub Trending AI/ML 周报引擎。实时追踪热门开源项目，AI 摘要 + 评论互动 + 个性化推荐，支持中文/英文/日文。

## 应用介绍
### 🚀 AI 潮汐 — 你的开源项目雷达

AI 潮汐 DevPulse 是一款专注 AI/ML 领域的开源项目发现工具，帮你追踪 GitHub 上最受关注的 trending 项目。

### 主要功能
- **实时 Trending**: GitHub/GitLab/Gitee 三源 trending
- **AI 智能摘要**: 每个项目自动生成中文摘要和核心要点
- **Star 趋势图**: 直观展示项目热度变化
- **评论与点赞**: 和全球开发者交流项目见解
- **个性化推荐**: 基于浏览偏好发现相关项目
- **离线可用**: PWA 支持，弱网环境也能使用
- **多语言界面**: 中文/English/日本語
- **隐私保护**: 不跟踪用户，数据加密

### 适用人群
- AI/ML 从业者和研究者
- 开源项目贡献者
- 技术团队负责人
- 关注技术趋势的开发者
