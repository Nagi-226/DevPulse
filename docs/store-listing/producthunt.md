## 上线标题

## 描述 (260 字)
DevPulse is your weekly radar for AI/ML open source innovation. We crawl GitHub Trending every week, generate Chinese/English AI summaries for each project, and help you discover the most exciting repos with personalized recommendations.

**Why DevPulse?**

- 🔥 **3-source trending**: GitHub + GitLab + Gitee, filtered by 20+ languages
- 🤖 **AI summaries**: DeepSeek-powered summaries extract key points from READMEs
- 💬 **Developer discussions**: Comment and like repos, share findings with peers
- 🧠 **Smart recommendations**: Collaborative filtering based on your browsing behavior
- 📊 **Star trends**: Visualize project growth over 30 days
- 🌍 **Multilingual**: Chinese, English, Japanese UI
- 📱 **Install anywhere**: PWA for mobile, Tauri for desktop, WebView for HarmonyOS
- 🔒 **Privacy-first**: No trackers, encrypted data, open source

Built by developers, for developers. Join 500+ early adopters!
