## 完整描述
### 🚀 发现 AI 领域最火的开源项目

AI 潮汐 DevPulse 帮你追踪 GitHub 上最受关注的开源项目，特别是 AI/ML 领域。每周自动生成 trending 周报，AI 生成中文摘要，让你快速了解每个项目是做什么的。

### ✨ 核心功能

- **🔥 实时 Trending**: 查看 GitHub/GitLab/Gitee 上 daily/weekly/monthly 热门项目
- **📊 数据可视化**: 每个项目的 Star 增长趋势曲线
- **🤖 AI 摘要**: LLM 自动生成项目摘要、核心要点和技术标签
- **💬 评论互动**: 对感兴趣的项目发表评论，和其他开发者交流
- **👍 点赞收藏**: 一键收藏和点赞，跨设备同步
- **🧠 个性化推荐**: 基于你的浏览和收藏，推荐相关项目
- **📱 PWA 支持**: 可安装到桌面，离线也能查看缓存内容
- **🌍 多语言**: 支持中文、English、日本語

### 🔒 隐私优先

- 不使用任何跟踪器
- 数据加密传输
- 不向第三方出售数据
- 开源透明
